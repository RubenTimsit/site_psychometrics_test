Suppression des données – Application Psycho

Les utilisateurs peuvent demander à tout moment la suppression de leur compte et de toutes leurs données personnelles associées.

Méthode de suppression

Pour supprimer votre compte, veuillez nous envoyer un e-mail à contact@psycho-app.com avec l’adresse utilisée pour l’inscription. Votre compte sera supprimé sous 7 jours ouvrables.

Conséquences

Une fois la suppression effectuée, toutes vos données (statistiques, favoris, historique) seront définitivement supprimées de notre base de données Firebase.
